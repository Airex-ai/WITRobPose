         WITRobPose is a set of attitude data set for six axis industrial robots, which contains 8477 labeled industrial robot images. Robot brands include Yaskawa, Fanuc, Kawasaki, etc. the operation scenarios include welding, grinding, cutting, disassembling, loading and unloading, handling, etc. the industries involved include car factories, etc. WITRobPose uses the method of manual annotation to mark the coordinate points of each axis joint of industrial robot, which can be used as the training data set and related research work of the attitude neural network model of industrial robot.
