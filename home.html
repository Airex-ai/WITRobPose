Introduction to datasets：

①Annotation method:
         Annotation method: we put the preprocessed pictures in the IMG folder, and then annotate one by one with dot code, and then dot the running pictures. Since our data sets are all six axis industrial robots, we mark each joint point (i.e., six joint points), and then run the connection code to connect to get the final labeled picture. If the line is outside the robot, adjust the position of the point until the robot's attitude can be displayed.
②How to use:

1.If there is video data, put the photo and video (through video_cut.py, the video file is placed in the same directory, the file name is changed to test.mp4, and the suffix in the code is changed if the suffix is different), select the appropriate frame parameters to intercept the video, and manually select the appropriate samples to store in the image for the difference sample elimination.

2.Arrange the images in the images folder, and then save the generated image folder of a video into the IMG folder by resize_img.py. If you need to continue adding videos, you need to set the img_new_name parameter to continue adding images later. Otherwise, the script above will overwrite the previous images.

3. If the image data is collected directly, put the image directly into the IMG folder, and then carry out the following operations.

4. Click mouse_callback.py for img folder, and draw_line.py line to save it into ANN folder.

5. Convert txt2json.py to JSON file, and check.py for JSON format detection. If an error is reported, check the online JSON file format on the official website of bejson to find the error and correct it. Generally, there is one and more comma problems.

Note: if the script has absolute path changed to relative path, the path will not report an error.
